# Gulp Scss Starter

#### To use this starter:

    git clone https://github.com/crstnmac/gulp-scss-starter.git && cd gulp-scss-starter && npm install

#### To run the server:

    gulp watch

##### Linux Mint users:

If you have an error called `autoreconf not found!!`.
`Install autoreconf`

    sudo apt-get install dh-autoreconf
